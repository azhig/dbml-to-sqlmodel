# 🚀 Quick Start

## Установка

```bash
# 1. Клонировать репозиторий
git clone <your-repo-url>
cd dbml_to_crud

# 2. Установить зависимости
make install
# или: uv sync
```

## Запуск интерактивного CLI

```bash
# Запустить интерактивное меню
uv run dbml-crud

# Или через Makefile
make cli
```

```bash

```

## Типичный рабочий процесс

```bash
# 1. Создать/изменить DBML схему
vim schemas/schema.dbml

# 2. Запустить CLI
uv run dbml-crud

# 3. В меню выбрать:
#    📊 Report (опционально) - посмотреть файлы и несоответствия
#    🎯 DBML → Code - сгенерировать

# 4. CLI автоматически:
#    - Покажет diff если есть изменения
#    - Запросит подтверждение
#    - Защитит USER_MODIFIED файлы
#    - Сгенерирует приложение

# 5. Запустить приложение
cd output
uv run uvicorn main:app --reload --port 8001

# 6. Открыть браузер
#    http://localhost:8001/docs   - API документация
#    http://localhost:8001/admin  - Админ панель
```

## Основные возможности

### 🎯 Интерактивное меню
- Навигация стрелками ↑↓
- Выбор Enter
- Отмена Ctrl+C

### 👁️ Автоматический Preview
При генерации CLI **автоматически показывает diff** перед применением изменений.

### 🔒 Защита кастомных файлов
Добавьте в начало файла:
```python
# USER_MODIFIED
```
И он будет автоматически защищен от перезаписи.

### 🔄 Continuous Workflow
После каждой операции можно вернуться в меню и выполнить следующую.
