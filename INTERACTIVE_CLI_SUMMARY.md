# 🎯 Interactive CLI - Summary of Changes

## Что было сделано

### ✅ Создан полноценный интерактивный CLI

Теперь вместо команд с аргументами, CLI работает как интерактивное приложение с меню.

### Основные изменения

#### 1. Новая зависимость
- Добавлена библиотека **questionary** для интерактивных меню

#### 2. Полностью переписан CLI ([src/dbml_to_crud/cli.py](src/dbml_to_crud/cli.py))

**Было:**
```bash
uv run dbml-crud generate schemas/schema.dbml -o output
```

**Стало:**
```bash
uv run dbml-crud
# Интерактивное меню с навигацией стрелками
```

#### 3. Реализованные функции

##### 🎯 Интерактивное меню
- Навигация стрелками ↑↓
- Выбор Enter
- 5 пунктов меню:
  - ⚙️ Настройки
  - 🎯 DBML → Code
  - 🔁 Code → DBML
  - 📊 Report
  - ❌ Exit - выход

##### 🤖 Автоматический Preview при генерации
**Ключевая фича!**

Когда вы выбираете "DBML → Code", CLI:
1. Проверяет существует ли директория вывода
2. Если да - **автоматически показывает diff**
3. Показывает статус всех файлов
4. **Запрашивает подтверждение**: "Применить изменения?"
5. Только после подтверждения генерирует

Это именно то, что вы просили!

##### ✅ Подтверждения по делу
- Применить изменения? (если директория вывода уже существует)

##### 🔄 Continuous Workflow
После каждой операции происходит возврат в меню.

```bash

```

1. **Первая генерация:**
   ```

   → Используются настройки из .dbml_to_crud
   → Готово!
   ```

2. **Обновление с просмотром изменений:**
   ```
   uv run dbml-crud
   → Выбрать "🎯 Генерация приложения"
   → Используются настройки из .dbml_to_crud

   → Y (применить изменения)
   ```

3. **Просмотр файлов и несоответствий:**

## Технические детали

### Архитектура

```
cli.py
├── interactive_menu()          # Главное меню
├── handle_settings()          # Настройки
├── handle_generate()          # DBML → Code
│   ├── Проверка директории
│   ├── Автоматический preview ⚡
│   └── Подтверждение
├── handle_code_to_dbml()      # Code → DBML
└── handle_report()            # Report
```

### Ключевая функция: Auto-Preview

```python
def handle_generate():
    # ...

    # First, show preview if output directory exists
    if output_dir.exists():
        console.print(Panel(
            "[yellow]Директория вывода уже существует.[/yellow]\n"
            "Сначала показываем предпросмотр изменений...",
            border_style="yellow",
            title="⚠️  Внимание",
        ))

        # Show preview
        preview.preview_command(...)

        # Ask for confirmation
        if not questionary.confirm("Применить изменения?", default=False).ask():
            console.print("[yellow]Генерация отменена[/yellow]")
            return

    # Continue with generation...
```
